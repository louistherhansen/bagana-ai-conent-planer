### Priority 1: Essential Assets
1. **Logo** - Verify `public/bagana-ai-logo.png` exists and is high-res
2. **Voiceover** - Record or source professional voiceover
3. **Screenshots** - Capture all platform views at 1920x1080
4. **Background Music** - Select royalty-free track

### Priority 2: Production Setup
1. **Choose Production Method:**
   - DIY: Use free tools (OBS, DaVinci Resolve, Audacity)
   - Semi-Pro: Hire voiceover artist, DIY editing
   - Professional: Full production service

2. **Set Up Workspace:**
   - Create `video/assets/` folder for raw materials
   - Create `video/exports/` folder for final outputs
   - Organize by scene (scene1/, scene2/, etc.)

### Priority 3: Quality Checks
1. **Audio:**
   - Clear, no background noise
   - Proper levels (not too loud/quiet)
   - Sync with visuals

2. **Visuals:**
   - Sharp, in focus
   - Consistent branding
   - Readable text overlays

3. **Timing:**
   - Exactly 60 seconds (±2 seconds acceptable)
   - Smooth transitions
   - No awkward pauses

## Potential Issues & Solutions

### Issue 1: Missing Platform Screenshots
**Solution:** 
- Use mockups/placeholders if UI isn't ready
- Create simple wireframes showing key features
- Use stock footage with overlays

### Issue 2: Voiceover Quality
**Solution:**
- Use AI voice synthesis (ElevenLabs) as backup
- Record multiple takes and choose best
- Use professional voiceover service if budget allows

### Issue 3: File Size Too Large
**Solution:**
- Compress video using HandBrake or FFmpeg
- Reduce bitrate if needed (minimum 5 Mbps)
- Consider creating multiple versions (web vs. download)

### Issue 4: Brand Colors Not Defined
**Solution:**
- Check `tailwind.config.js` for brand colors
- Use teal/blue gradient as default
- Ensure consistency across all visuals
